## Running Locally

- Install [Node](http://nodejs.org/download/) and [Grunt](http://gruntjs.com/)
- Install the dependencies

```
npm install
```

Then just type on your shell:

```
grunt connect
```

Open your browser at `http://localhost:8001/`
